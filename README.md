# montage_calc

## Установить зависимости
```
npm install
```
### Запуск тестового сервера
```
npm run server
```
### Запуск проекта
```
npm run serve
```
P.S.
если нет IDE можно воспользоваться двумя cmd консолями(указав путь до корня проекта)
(обычно прослушивание порт 8080,но может быть и другой)
